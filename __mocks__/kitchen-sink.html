Benefits to building with Project Helix

  • Content at scale
    Empower more teams to build their own content. Build your site with a set of reusable building blocks that are always on brand and you're in control of.
  • Web content management
    Preview content before it goes live with Project Helix Sidekick. Get a birds eye view at the touch of a button with Project Helix Organizer. Use familiar tools like Microsoft SharePoint or Google Drive to manage content permissions.
  • Personalized experiences
    Create a unified customer profile spanning the breadth and depth of your data, and then use it to deliver the most relevant experience for each customer in meaningful, memorable ways. Get details
  • Code simplified
    Say goodbye to complex APIs spanning multiple languages. Say hello to simple content structures decoupled from code. Anyone with a little bit of HTML, CSS, and JS can build a site on Project Helix.
  • Headless is here
    Go directly from Microsoft Excel or Google Sheets to the web in mere seconds. Sanitize and collect form data at extreme scale with Project Helix Forms.
  • Peak performance
    Use Project Helix's serverless architecture to meet any traffic need. Use Project Helix's PageSpeed Insights Github action to evaluate every commit for Core Web Vitals.

How it works

Create content in Microsoft Word or Google Docs.

Create a new folder in Google Drive or Microsoft SharePoint. Then create a new document and put some content in it.

Share your folder with Project Helix.

After you create a page, head back to your folder and share it with Helix.

Give your site some features.

Create a new repository in Github. Add styles and scripts to make your site completely bespoke to your own needs.

See your site in action.

After you have added some content, shared with Project Helix, built a repo, and tied your repo to your conent, you can see your site in action.
